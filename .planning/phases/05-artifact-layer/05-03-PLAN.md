**In artifacts.rs**, add a public function:

```rust
/// Process existing tool_executions to retroactively populate artifact tables.
///
/// This handles data ingested before the artifact layer (migration 003) existed.
/// Queries tool_executions joined with messages, processes each in chronological
/// order, and inserts file_operations/git_operations/files rows.
///
/// Idempotent via INSERT OR IGNORE -- re-running on already-decomposed data
/// produces no duplicates. [ART-04 retroactive support]
pub fn decompose_artifacts_retroactive(
    conn: &Connection,
) -> Result<usize, DecomposeError>
```

Implementation:
1. Query all tool_executions with their message context:
   ```sql
   SELECT te.tool_use_id, te.tool_name, te.input_json, te.result_content, te.is_error,
          m.uuid, m.session_id, m.timestamp
   FROM tool_executions te
   JOIN messages m ON m.uuid = te.message_uuid
   WHERE te.tool_name IN ('Write', 'Edit', 'Read', 'Bash')
   ORDER BY m.timestamp ASC
   ```
2. Open an unchecked_transaction (following PAT-010)
3. For each row, parse input_json as serde_json::Value and dispatch to the same extraction functions used by decompose_artifacts (extract_write_operation, etc.)
4. All INSERT OR IGNORE ensures idempotency
5. Commit transaction, return total rows inserted

Note: The extraction functions already take (session_id, tool_use_id, message_uuid, timestamp, input: &Value, tx: &Transaction). The retroactive function constructs these parameters from the query results and calls the same functions.

**In sync.rs**, after sync_all completes the file-by-file sync loop and before returning SyncAllResult:

1. Add a call to `crate::artifacts::decompose_artifacts_retroactive` inside a conn.call closure
2. Log the number of retroactive artifact rows at info level
3. This runs AFTER all files are synced, so all tool_executions rows exist before retroactive processing begins
4. Wrap in a separate call so failures are logged but do not prevent sync_all from returning successfully

```rust
// Retroactive artifact decomposition: backfill files/file_operations/git_operations
// from existing tool_executions. Idempotent, safe to run on every sync.
if result.total_records > 0 {
    match conn.call(|conn| {
        crate::artifacts::decompose_artifacts_retroactive(conn)
            .map_err(|e| rusqlite::Error::ToSqlConversionFailure(Box::new(e)))
    }).await {
        Ok(rows) => tracing::info!(artifact_rows = rows, "Retroactive artifact decomposition complete"),
        Err(e) => tracing::warn!(error = %e, "Retroactive artifact decomposition failed (non-fatal)"),
    }
}
```
